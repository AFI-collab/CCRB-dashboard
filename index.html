How to update this dashboard weekly

1. Export from QBO — Run the P&L filtered to this job. Pull the revenue and each COGS line. Enter above.
2. Check your AIA — After submitting App #5, update PCONC1 this period and Period To date.
3. Log T&M days — Update how many of the 9 T&M days are complete.
4. Save — All charts, burn rate bars, and alerts recalculate instantly.
5. Host it — Upload this HTML file to GitHub Pages (free) or embed in GoHighLevel as an iframe for team access.
